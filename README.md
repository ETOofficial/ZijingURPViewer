# ZijingURPViewer

## **简介**

本应用是**面向学生**的 `南京理工大学紫金学院教务系统网站` 的学生信息查询工具。

目前实现的功能有：

- [x] 通过校外访问路径登录
- [ ] 通过校内访问路径登录
- [x] 查询考试安排
- [x] 查询考试成绩
- [ ] 其它未计划功能
